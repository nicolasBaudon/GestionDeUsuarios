# Modulo De Gestion de Usuarios/Clientes

## Este modulo es una aplicacion web desarrolada con JQuery, PHP y MySql, con la integracion de DataTables y Boostrap 4.

Consiste en un listado de Usuarios/Clientes, alojados en una base de datos, estos son tomados desde la misma y mostrados para su uso, en el caso que se desean imprimir o simplemente para tener un seguimiento de ellos.

---

En este caso la aplicacion esta desarrolada con animales, ya que en la base de datos se encuentra de esa manera y fue desarrollada para una veterinaria, pero esto puede ser cambiado desde la misma.
